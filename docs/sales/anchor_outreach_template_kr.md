# Anchor Outreach Template (KR)

> 1차 cold outreach용. LinkedIn DM, 산업 인맥 소개, 이메일 다 호환되는 짧은 버전.

## Template A — LinkedIn DM / 짧은 메시지 (200자 이내)

```
[이름] 님 안녕하세요. SMT AOI 가성불량(약 30%) 자동 소거 솔루션을
6개월 무료 파일럿으로 소개드리고자 메시지 드립니다. 벤더 무관(Saki·Koh
Young·Mycronic), 온프레 only, escape 0 보장 — 첫 3개 사이트는 비용
0원입니다. 30분 화상 가능하실까요?

aoi-sentinel — github.com/DrJinHoChoi/aoi-sentinel
```

## Template B — 이메일 (소개 받은 경우)

```
제목: [소개자 이름]님 통해 인사드립니다 — SMT AOI 가성불량 자동 소거
무료 파일럿 제안

[이름] 님 안녕하세요.

[소개자 이름]님 추천으로 연락드립니다.

저는 SMT AOI 후공정의 가성불량(false call, 약 30%)을 운영자 결정 학습을
통해 자동으로 줄이는 시스템을 개발 중입니다. 핵심 차별점:

  • 벤더 무관 (Saki / Koh Young / Mycronic / TRI 등 모두)
  • 온프레 only — 데이터 외부 유출 없음
  • Escape 하드 제약 — 진성불량 누출 시 자동 강등
  • 2차 검사 → 라인 → 사이트 단위로 점진 확장

현재 첫 3개 anchor 사이트에 한해 **6개월 무료 파일럿**을 운영합니다.
효과 미달 시 비용 0원, 위험 없이 시작하실 수 있습니다.

핵심 자료:
  • Pilot 1-pager (PDF): [링크]
  • Open standard RFC: github.com/DrJinHoChoi/aoi-common-spec
  • Reference implementation: github.com/DrJinHoChoi/aoi-sentinel

30-45분 화상 미팅 가능하실까요? 다음 주 화·수·목 오후 중 편하신 시간
알려주시면 맞추겠습니다.

감사합니다.

— 최진호 (DrJinHoChoi)
```

## Template C — 미팅 후속 (LOI 권유)

```
[이름] 님 안녕하세요.

오늘 미팅 감사했습니다. 논의 내용 정리하면:

  • 귀사 [라인 이름]에서 AOI 가성불량률 약 [X]% 추정
  • 일 처리량 ~[Y] 보드, 운영자 [Z]명이 2차 검사
  • 6개월 파일럿 적합 — SHADOW 모드로 라인 영향 0
  • 셋업 가능 시점: [Date]

다음 단계로 LOI(파일럿 의향서) 1페이지를 첨부합니다. 법무 검토 거쳐
서명 부탁드립니다. 셋업까지 약 2주 소요되며, 첫 30일은 베이스라인
측정만 합니다.

LOI 포인트:
  • 파일럿 비용: ₩0
  • 기간: 6개월
  • 성공 기준: 가성불량 70%↓, escape 0, 운영자 200hr/월 절감
  • 미달 시: 자동 종료, 비용 0
  • 데이터 사용: 6개월 간 운영 데이터 학습 사용권 + 케이스 스터디 동의

질문 있으시면 언제든 답주세요.

— 최진호
```

## 사용 가이드

1. **A → B → C 순서로 escalate**. A는 처음 닿기, B는 미팅 잡기, C는 클로즈.
2. **각 메시지 길이 엄수**. 짧을수록 회신율 ↑.
3. **첨부는 1-pager만**. 슬라이드 X. 결재선이 짧을수록 의사결정 빠름.
4. **회신 없을 때 follow-up 1회만**. 2주 후 1줄 reminder. 그 이후엔 cold list로 보내고 다른 후보로 이동.
5. **거절 시 거절 이유 1줄로 받기**. "왜 안 되나요?" 질문 한 번. 데이터로 다음 후보에 반영.
