## 슬라이드 1 — 표지

### 🎯 공장에서 매일 사람이 하는 검사 일 — AI가 도와드립니다.

```
              aoi-sentinel
        — 공장의 작업자를 돕는 AI 박스 —
            DrJinHoChoi · 2026
```

**💡 한 줄 정리**: 작은 박스를 공장에 놓아두면, 사람이 매일 하는 반복 검사 일 중 **70%를 대신해주는 AI**입니다.

---

## 슬라이드 2 — 공장에서 매일 일어나는 일

### 🎯 자동차 부품 공장에서는 매일 작은 부품 100만 개를 검사합니다

```
[자동 검사기]  →  "이 부품 불량이에요!" (NG 표시)
                          ↓
[사람 작업자]  →  "어디 봅시다... 음... 맞네" 또는 "아닌데?"
                          ↓
                  결정: 폐기 / 재가공 / 통과
```

**💡 비유**: 공항에서 X-ray 검사기가 가방을 빨간색으로 표시하면, 사람이 다시 확인하는 것과 같습니다.

---

## 슬라이드 3 — 문제

### 🎯 검사기가 "불량"이라 한 것 100건 중, 진짜 불량은 30건뿐. 나머지 70건은 헛수고.

```
검사기가 "불량" 표시  100건
   ├─ 진짜 불량         30건  ← 사람이 확인 후 폐기
   └─ 사실은 멀쩡        70건  ← 사람이 확인 후 그냥 통과
                                  (시간 낭비)
```

→ 작업자분은 **매일 70번의 헛수고**를 하고 계십니다.

**💡 의미**: 매일 야근, 라인 처리량 감소, 작업자 피로 — 모두 이 70번의 헛수고에서 옴.

---

## 슬라이드 4 — 우리가 하는 일

### 🎯 검사기 옆에 작은 박스를 놓으면, AI가 작업자처럼 확인을 도와줍니다.

```
[검사기] → [aoi-sentinel 박스] → 결과 셋 중 하나:

           ✓ "이건 분명 멀쩡합니다. 통과시키세요"   (자동)
           ✗ "이건 진짜 불량이네요. 폐기하세요"     (자동)
           ? "음... 잘 모르겠어요. 봐주세요"        (작업자에게)
```

→ 작업자는 **확실하지 않은 것만** 보면 됩니다.

**💡 비유**: 학교에서 시험 채점을 보조 선생님이 1차 채점해주고, 헷갈리는 것만 본 선생님에게 가져오는 것과 같습니다.

---

## 슬라이드 5 — 박스가 어떻게 똑똑해지나

### 🎯 매일 작업자가 결정하는 걸 보고, 박스가 스스로 배워나갑니다.

```
Day 1:    박스: "음... 잘 모르겠어요" (100건 중 100건 escalate)
          작업자: 100건 다 확인 → 박스에게 정답 알려줌
                                          ↓
Day 30:   박스: "이런 건 알겠어요!" (100건 중 70건 자동)
          작업자: 30건만 확인
                                          ↓
Day 180:  박스: "이제 거의 다 알아요" (100건 중 95건 자동)
          작업자: 5건만 확인
```

**💡 비유**: 신입사원이 6개월간 사수 옆에서 일 배우는 것과 같습니다. 처음엔 다 물어보지만 점점 혼자 할 수 있어집니다.

---

## 슬라이드 6 — 절대 안 일어나는 일 — 안전 약속

### 🎯 진짜 불량을 놓치는 일은 절대 없습니다. 수학으로 보장합니다.

```
박스가 모르겠으면:
        →  무조건 작업자에게 물어봄 (escalate)
        →  그 결정은 작업자가 함
        
→ 진짜 불량을 박스가 혼자 "통과"시키는 사고는 발생 안 함.
```

**작업자 + 박스 = 둘 중 하나는 반드시 확인.**  **둘 다 못 본 사고 = 0건.**

만약 단 한 번이라도 사고가 발생하면 → 박스는 **자동으로 보수적 모드로** 돌아가서 모든 결정을 다시 작업자에게 넘김.

**💡 비유**: 자동차 자율주행이 헷갈리면 운전자에게 핸들 넘기는 것과 같습니다. 안전이 최우선.

---

## 슬라이드 7 — 3단계로 천천히 도입

### 🎯 처음 한 달은 박스가 일 안 합니다. 보기만 해요. 작업자분이 적응할 시간.

```
1단계 (한 달):     박스가 의견만 표시. 작업자 100% 결정.
                   → 안전 검증

2단계 (5-6개월):   박스가 확실한 것만 자동. 모르면 작업자에게.
                   → 작업자 부담 점점 감소

3단계 (이후):       박스가 거의 다. 작업자는 KPI만 모니터링.
                   → 옵션 (회사가 결정)
```

**💡 비유**: 새 직원에게 처음엔 옆에서 보게 하고, 익숙해지면 쉬운 일부터 맡기고, 마지막엔 알아서 일하게 하는 것과 같습니다.

---

## 슬라이드 8 — 6개월 무료 시범 도입

### 🎯 6개월 무료로 써보세요. 효과 미달이면 비용 0원, 박스 회수.

| 항목 | 내용 |
|------|------|
| **비용** | **0원** (박스 + 셋업 + 지원 모두 무료) |
| **기간** | 6개월 |
| **약속** | 가성불량 70% 줄지 않으면 자동 종료 |
| **위험** | 0 — 라인 운영에 영향 없음 (1단계는 보기만) |

**💡 의미**: 마트에서 시식 코너 같은 거예요. 맘에 안 들면 그냥 두고 가시면 됩니다.

---

## 슬라이드 9 — 어떤 검사기든 됩니다

### 🎯 일본 Saki든, 한국 Koh Young이든, 어느 회사 검사기든 다 됩니다.

```
[Saki 검사기] ─┐
[Koh Young]   ├─→  같은 박스 하나가 다 받아들임
[Mycronic]    │
[기타 vendor] ─┘
```

라인에 두세 종류 검사기가 섞여있어도 **박스 하나로 통합**.

**💡 비유**: 어떤 회사 휴대폰이든 USB-C 케이블 하나로 충전 가능한 것과 같습니다 (벤더 무관 표준).

---

## 슬라이드 10 — 데이터는 공장 밖에 안 나갑니다

### 🎯 회사의 검사 데이터는 공장 안에서만 사용. 클라우드 전송 없음. 외부 회사 모름.

```
[검사 데이터] → [공장 안 박스] → 학습 → 더 똑똑해진 박스
                       │
                       └── 외부로 전송 0건
```

→ 자동차 부품·반도체·기술 비밀이 새지 않습니다.

**💡 의미**: 가족 비밀 일기를 집 밖으로 안 가져가는 것과 같습니다. 박스는 공장 안에서만 일합니다.

---

## 슬라이드 11 — 우리만 못 따라하는 것

### 🎯 다른 회사가 AI 모델 따라 만들 수는 있지만, "표준" 자체는 못 만듭니다.

```
다른 회사:   "우리도 AI 만들었어요"  (1-2년이면 따라옴)
                          ↓
우리:        "AI들이 따라야 할 표준 자체를 정의 중입니다"
                          ↓
              5년 후엔 정부가 의무화 (EU·한국 AI법)
                          ↓
              모든 회사가 우리 표준 통과해야 검사 AI 팔 수 있음
```

**💡 비유**: Bluetooth 만든 회사 하나가 있어요. 다른 회사들이 Bluetooth 따라 만들 수는 있지만, **"Bluetooth라고 부를 권한"은 처음 정의한 곳**에 있습니다. 우리가 그 자리.

## 슬라이드 12 — 더 큰 그림

### 🎯 같은 박스가 SMT 검사뿐 아니라 자동차·로봇·AI 서버 다 적용됩니다.

```
오늘:     자동차 부품 검사     (PCB 보드)
          ↓
1년 후:   AI 서버 보드 검사    (NVIDIA, HBM)
          ↓
2년 후:   휴머노이드 부품      (Tesla Optimus 같은 것)
          ↓
3년 후:   EV 배터리 검사       (전기차)
          ↓
5년 후:   글로벌 표준 layer    (모든 제조 AI)
```

**💡 의미**: 우리가 잘 만든 박스 하나가 **모든 제조업의 미래**에 들어갑니다.

---

## 슬라이드 13 — 5년 후 — 정부가 의무화

### 🎯 EU·한국·미국 모두 2027-2030년에 "AI 검사 의무 audit" 도입합니다.

```
2024  EU AI 법 발효
2025  한국 AI 기본법 통과
2026  ISO 국제 표준 가속
2027  ↓ 의무화 시작
2030  글로벌 AI audit 의무화
```

→ 그 시점에 **AI 솔루션은 표준 통과해야 판매 가능**. 우리가 그 표준 정의자.

**💡 의미**: 자동차에 안전벨트 의무화된 것처럼, AI도 5년 후엔 인증된 것만 사용 가능. 우리는 그 인증 기관.

## 슬라이드 14 — 실제 결과 — VisA 실험

### 🎯 컴퓨터에서 60번 학습 → 진짜 불량 놓침 0건, 작업자 부담 5배 감소.

```
┌──────────────────┬──────────────────┐
│ 놓친 불량         │   작업자 부담     │
│      0건         │     5배 감소      │
│   (60번 학습)    │  (1280 → 230)    │
└──────────────────┴──────────────────┘
```

→ 이건 컴퓨터 실험. 실제 공장에선 같은 일이 **6개월 단위**로 일어남.

**💡 의미**: 박스가 60번 시험 보면서 **단 한 번도 진짜 불량을 놓치지 않았습니다**. 그 사이 작업자 부담은 5배 줄었음.
