# 파일럿 의향서 (Letter of Intent — Pilot)

> **본 문서는 법적 구속력이 있는 본 계약 체결 전 단계의 의향서이며, 양 당사자의 파일럿 운영 의사를 확인하기 위한 것입니다. 본 LOI 자체는 제5조에 명시한 사항을 제외하고는 법적 구속력을 갖지 않습니다.**

---

## 제1조 (당사자)

| 구분 | 내용 |
|------|------|
| **공급자** | DrJinHoChoi (이하 "공급자") — aoi-sentinel 시스템 개발사 |
| **고객사** | _________________________ (이하 "고객사") |
| **체결일** | 20___ . ___ . ___ |
| **유효기간** | 체결일로부터 14일 이내 본 파일럿 셋업 일정 확정 |

## 제2조 (목적)

본 LOI는 공급자가 보유한 **aoi-sentinel** (벤더 무관 SMT AOI 가성불량 자동 소거 시스템)을 고객사 SMT 라인에 6개월간 무상 시범 운영하여, 그 효과를 측정하고 본 계약 체결 여부를 결정하기 위한 양 당사자의 의향을 명시함을 목적으로 합니다.

## 제3조 (파일럿 범위 및 일정)

| 항목 | 내용 |
|------|------|
| **대상 라인** | 고객사 ____________ 공장 ____________ 라인 1개 (이하 "대상 라인") |
| **AOI 벤더** | _____________ (Saki / Koh Young / Mycronic / 기타) |
| **파일럿 비용** | **₩0** — 하드웨어·소프트웨어·셋업·교육·6개월 지원 일체 무상 |
| **운영 모드** | SHADOW 모드 (라인 운영에 영향 없는 보조 모드) |
| **셋업 기간** | LOI 체결 후 14일 이내 |
| **기간** | 셋업 완료일로부터 6개월 |

### 단계별 일정

```
D 0       LOI 체결
D 0-14    셋업 (Edge box 배송, AOI 결과 폴더 read-only 마운트, 운영자 교육)
D 15-44   30일 베이스라인 측정 (현재 가성불량률 정량화)
D 45-180  SHADOW 운영 — 라벨 누적, 모델 자동 개선
D 180     성과 평가 및 본 계약 체결 여부 결정
```

## 제4조 (성공 기준)

본 파일럿은 셋업 완료 후 6개월 시점에 다음 3개 지표를 모두 달성한 경우 성공으로 판정합니다.

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| **가성불량 감소율** | **70% 이상** | 베이스라인 30일 vs 종료 시점 30일, 동일 라인 동일 제품 기준 |
| **escape (진성불량 누출)** | **0건** | 운영자 재검 라벨 + 후속 공정에서의 누출 추적 |
| **운영자 부담 감소** | **월 200시간/라인 이상** | 2차 검사 소요 시간 측정 |

**성공 기준 미달 시**: 본 파일럿은 자동 종료되며, 고객사는 어떠한 비용도 부담하지 않습니다. 공급자는 박스를 회수하고, 고객사 요청 시 누적 데이터 일체를 파기합니다.

## 제5조 (양 당사자의 의무 — 구속력 있음)

본 조항은 본 LOI 중 유일하게 법적 구속력을 갖는 부분이며, 양 당사자는 다음을 보증합니다.

### 5.1 공급자의 의무

- (a) 셋업·운영·6개월 지원 일체를 무상으로 제공한다.
- (b) 박스는 SHADOW 모드에서만 동작하며, 고객사 라인 운영을 자동으로 변경하지 않는다.
- (c) 누적된 검사 데이터·이미지·라벨은 **고객사 물리적 시설 외부로 반출하지 않는다** (온프레 only).
- (d) 본 파일럿 종료 시점에 고객사가 요청하면 누적 데이터 일체를 파기한다.

### 5.2 고객사의 의무

- (a) 대상 라인의 AOI 결과 폴더에 대한 **read-only 접근**을 공급자에게 제공한다.
- (b) 운영자 1명이 일 평균 **30분 이내**의 시범 결정 검토 시간을 할당한다.
- (c) 6개월 운영 데이터에 대한 **공급자의 학습 사용권**을 부여한다 (단, 사용은 고객사 시설 내부에 한정).
- (d) 본 파일럿 결과를 **익명화된 케이스 스터디**로 공급자가 외부에 공개하는 것에 동의한다 (회사명·라인명·제품명은 마스킹).

### 5.3 비밀 유지 (NDA)

본 LOI 체결과 동시에 양 당사자 간 비밀유지 의무가 발생한다. 공급자는 고객사의 제품·라인·운영 데이터를 외부에 공개하지 않으며, 고객사는 공급자의 시스템 내부 동작·파라미터·소스코드 비공개 부분을 외부에 공개하지 않는다. 본 의무는 본 LOI 종료 후 **5년**간 유효하다.

## 제6조 (본 계약 체결 시 가격 — 비구속)

성공 기준 충족 시 양 당사자는 다음 중 하나의 본 계약을 체결할 수 있다 (단, 가격은 협의로 조정 가능).

| 옵션 | 조건 |
|------|------|
| **A. 영구 라이선스** | 사이트당 ₩100M-200M (라인 무제한) + 연 지원료 라이선스가의 15% |
| **B. 연구독** | 라인당 ₩30M-50M/년 (HW 임대 + SW + 지원 일괄) |
| **C. 추가 파일럿** | 다른 라인·다른 벤더에 대해 동일 조건 무상 파일럿 |

본 LOI 체결 시점에 옵션을 결정할 필요는 없으며, 6개월 평가 시점에 양 당사자가 협의하여 결정한다.

## 제7조 (조기 종료)

다음의 경우 양 당사자는 서면 통보로 본 LOI를 즉시 종료할 수 있다.

- (a) 박스가 고객사 라인 운영에 직접적 장애를 야기한 경우
- (b) 데이터 보안 사고가 발생한 경우 (공급자 측 책임 한정)
- (c) 양 당사자 합의에 의한 종료
- (d) 일방의 본 LOI 위반 사실이 30일 이상 시정되지 않는 경우

조기 종료 시 공급자는 박스를 회수하며, 고객사는 어떠한 비용도 부담하지 않는다.

## 제8조 (책임 제한)

본 파일럿은 SHADOW 모드로 운영되어 라인 운영에 직접 영향을 미치지 않으므로, 공급자는 고객사 제품 품질 또는 라인 가동에 대해 직접적 책임을 지지 않는다. 단, 공급자의 고의 또는 중과실로 인한 손해는 본 조항의 적용을 받지 않는다.

## 제9조 (준거법 및 분쟁 해결)

본 LOI는 대한민국 법령에 따라 해석되며, 본 LOI에 관한 분쟁은 양 당사자의 합의에 의해 해결하되, 합의에 이르지 못하는 경우 서울중앙지방법원을 제1심 관할법원으로 한다.

## 제10조 (효력)

본 LOI는 양 당사자의 서명일로부터 효력을 발생하며, 양 당사자가 동일한 효력을 가진 사본 2부를 보관한다.
