# 데모 플레이북 (KR)

> **5분 안에 anchor 미팅에서 박스를 동작시키고 LOI까지 끌고 가는 대본.**
> 외울 필요는 없다. 이 흐름대로 박스가 자동으로 돌아간다.

---

## 사전 준비 (미팅 1시간 전)

- [ ] 박스 부팅 1회 (배터리 풀충 확인)
- [ ] Wi-Fi 핫스팟 또는 회의실 LAN으로 IP 잡혀있는지 확인 (`ip a`)
- [ ] `xdg-open http://localhost:8080` 으로 UI 정상 표시 확인
- [ ] `~/demo_bundle` 새로 생성 (`make_demo_bundle.py --n_boards 50 --seed $(date +%s)`)
- [ ] 인쇄물 3종 휴대: pilot 1-pager + LOI + 명함
- [ ] 휴대폰 충전 90%+ (실시간 QR 스캔용)

## 고객 반응별 대응

### "흥미롭긴 한데 검토 시간이 필요합니다"

```
[당신]   "당연합니다. 1-pager + LOI 두 장만 가져가셔서 사내 의사결정자
          분께 보여주시면 됩니다. 14일 내 답주시면 첫 3개 사이트 안에
          들어가실 수 있습니다. 그 이후엔 ₩30-50M 시범가로 전환됩니다."
```

### "이미 [Mycronic / Koh Young AI] 검토 중입니다"

```
[당신]   "좋습니다 — 저희는 그걸 대체하는 것이 아니라 보완입니다.
          저희는 vendor-neutral이라 [회사명] 라인에 검사기가 여러
          벤더 섞여있다면 그분들 솔루션으론 한 종류만 커버됩니다.
          저희는 다 커버하고요. 그리고 [그 벤더] 솔루션은 자기네
          클라우드로 데이터 보냅니다. 저희는 안 보냅니다."
```

### "가격이 본 계약 시 ₩100M 수준이면 결재선이 깁니다"

```
[당신]   "맞습니다. 그래서 파일럿이 무료입니다. 6개월 운영 후
          가성불량 70% 줄었다는 측정값을 손에 들고 결재선에 들어가시는
          겁니다. 그 시점엔 ROI 수학이 명확해서 결재가 빠릅니다.
          ₩100M 라이선스가 라인 1개당 월 200시간 절감 = 연 ₩15-35억
          절감 → 2-4개월 회수입니다."
```

### "보안 우려 있습니다"

```
[당신]   "데이터는 박스 밖으로 안 나갑니다. LOI 5조에 명시되어 있고
          기술적으로도 박스에 outbound 인터넷 차단해두셔도 동작합니다.
          모델 업데이트도 SD 카드로 받아서 SneakerNet 가능합니다.
          저희가 본사 클라우드에 접근할 수 있는 백도어 같은 건 없습니다.
          소스코드 일부 공개되어 있으니 보안팀이 보안 감사 가능합니다."
```
